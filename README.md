# 📚 Galgame百科项目

**一个为 Galgame 与视觉小说爱好者打造的知识库**

主站：[https://www.galgame.it/wiki](https://www.galgame.it/wiki)

GitHub 仓库：[https://github.com/sombermengling/galgamewiki](https://github.com/sombermengling/galgamewiki)


## 项目概述

**Galgame百科项目**是一个由爱好者发起的非营利性开放知识库，专注于 Galgame（美少女游戏）与视觉小说（Visual Novel）领域的信息整理、知识归档与文化研究。

项目涵盖三个主要部分：

1. **Galgame Wiki** —— 系统化的百科资料库
2. **《Galgame研究》期刊** —— 深度学术研究
3. **开源协作社区** —— 让每个人都能参与建设

这不是一个人的项目，而是一个开放的共同体。我们相信，Galgame 作为一种独特的叙事媒介，值得被系统地记录、研究和传承。


## 诞生背景：一个玩家的十年存档实验

在 Galgame 圈子里，有一个长期存在的问题——信息太散了。

早期，玩家们依靠个人博客、贴吧、论坛来分享和讨论。但这些信息载体的生命周期太不稳定了：博客会停更，论坛会关站，贴吧的帖子会被系统清理。一个十年前的精品攻略帖，今天可能连搜索引擎都找不到。

更重要的是，这些信息之间互不相通。查游戏资料要去 A 站，找攻略要去 B 站，解决技术问题要去 C 论坛。一个玩家想要完整了解某部作品，不得不在多个平台之间来回跳转。

Galgame百科项目就是为解决这个问题而诞生的——把散落在各处的信息汇集到一起，让需要的人能一站式找到答案。


## 发起人

项目的发起人是 **忧郁的Dt君**

他是 Galgame 圈子里一位活跃了十多年的老玩家。早年做资源分享，后来参与汉化，再后来自己建站、做制作组，还曾参与制作“八月社纪念视频”。从资源分享者到知识库建设者，身份在变，内核从未改变。

2022 年，他在一则视频中写下了八个字：

> **“正因为热爱，所以坚信。”**

这句话后来被很多人记住。它不是一句口号，而是一个人做了十多年之后自然而然写出来的东西。

在 Galgame 文化仍然处于小众的当下，愿意为一个非营利的文化项目持续投入多年的人并不多。而忧郁的Dt君正是其中之一。


## 为什么要做这件事？

做 Galgame百科项目，原因很简单：

**第一，效率。** 一个能覆盖多数需求的资料库，比在十几个平台之间反复跳转省事得多。

**第二，保存。** 互联网不会自动保存任何东西。论坛会关，博客会停，帖子会删。有人专门做整理和归档，那些信息才能留下来。

**第三，门槛。** Galgame 圈子有很多“黑话”和文化背景，新人进来容易懵。有一个地方能系统地查询，入坑的门槛就降低了。

**第四，文化。** Galgame 发展了几十年，从引擎技术的演变到社群文化的变迁，这些东西值得被记录。不只是为了当下的玩家，也是为了以后还有人想看的时候，能够找到。


## Galgame百科在做什么？

### 资料库建设

目前站内覆盖了 Galgame 相关的多个维度：概念术语、工具引擎、游戏资料、会社档案、亚文化现象、争议话题、作品原型地、文化梗等。

每一个条目都力求克制的表述——不带情绪，只列事实。让资料回归资料本身。

### 学术探索

通过《Galgame研究》期刊，项目正在尝试用更严肃的方法理解 Galgame。目前已发表的论文涵盖了玩家画像、机器学习评分预测、经典作品叙事解读等多个方向。

从资料整理到学术研究，这个项目的版图正在一步步扩大。

### 开源协作

项目托管在 GitHub 上，采用开放协作的方式运作。任何人都可以参与编辑、提交建议、报告问题。项目坚持财务透明原则，年度运营支出公开，经费主要依靠用户赞助。


## 项目现状

经过几年的持续建设，Galgame百科项目已经积累了一定的基础。

据公开信息，目前已有多个 Galgame 相关站点在利用 Galgame Wiki 维护的游戏元数据。这意味着它正在从一个个人项目，逐步成为社区的基础设施。

也曾有研究者因为在 Wiki 里找到了别处查不到的资料，才得以完成学术论文。这些反馈，都是项目持续前行的动力。


## 如何参与？

参与的方式可以很简单：

- 修正一个错别字
- 补充一条发售日期
- 添加一个游戏封面图
- 完善一个会社的介绍
- 在讨论区提出建议

如果想做得更多：

- 整理你熟悉的游戏或会社词条
- 提供圣地的实景照片或坐标
- 翻译日文或英文资料
- 向《Galgame研究》期刊投稿


## 写在最后

Galgame 不只是游戏，它是一种叙事方式，是一段文化记忆。

Galgame百科项目想做的，就是让这些记忆不散落、不消失、不被遗忘。

欢迎来查资料，也欢迎来参与建设。让每一部优秀的作品，不仅被玩过，更被记录、被理解、被传承。
